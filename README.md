# 🎓 Online College Complaint Management System

A web-based complaint management system built using **Flask** that enables students to submit complaints online and allows administrators to efficiently manage and resolve them.

---

## 🚀 Features

### 👨‍🎓 Student
- Student Registration
- Student Login & Logout
- Dashboard
- Submit Complaint
- View My Complaints
- Edit Complaint
- Delete Complaint
- Complaint Status Tracking

### 👨‍💼 Admin
- Secure Admin Login
- Dashboard with Statistics
- Manage Categories (CRUD)
- Manage Complaints
- Update Complaint Status
- Manage Students
- View Complaint Details

---

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/online-college-complaint-management-system.git
```

### Go to the project folder

```bash
cd online-college-complaint-management-system
```

### Create a virtual environment

```bash
python -m venv venv
```

### Activate the virtual environment

Windows

```bash
venv\Scripts\activate
```

Linux / macOS

```bash
source venv/bin/activate
```

### Install dependencies

```bash
pip install -r requirements.txt
```

### Run the project

```bash
python run.py
```
